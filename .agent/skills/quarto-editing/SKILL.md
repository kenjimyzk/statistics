# Quarto Editing Skill

QMD (Quarto Markdown) ファイルのフォーマットを適切なMarkdown構文に修正します。

## ルール: リスト前の改行

qmdファイル内でリスト（箇条書き `*`, `-`, `+` や 番号付きリスト `1.`）を使用する際、**その直前の行が見出し（`#`）でない場合は、必ず空行を1行挿入してください。**

### 具体例

**悪い例（Bad）**:
テキスト本文です。
- リスト項目1
- リスト項目2

**良い例（Good）**:
テキスト本文です。

- リスト項目1
- リスト項目2

**例外（見出しの直後はOK）**:
### 見出し
- リスト項目1
- リスト項目2

## 実行指示
このスキルが呼び出された場合、対象のqmdファイルをスキャンし、上記のルールに違反している箇所（テキスト直後に改行なしでリストが開始されている箇所）を修正してください。
